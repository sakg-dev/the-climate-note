# The Climate Note

The Climate Note is a teen-led initiative that raises awareness about climate change by sharing daily questions and articles through email newsletters. Built by teens, for teens, it aims to make climate learning simple, engaging, and accessible.

This repository contains the official website that supports and showcases the initiative.

---

## Tech Stack

* Next.js
* TypeScript
* React
* Tailwind CSS

---

## Features

* Clean and responsive UI
* Informational website for the initiative
* Newsletter-focused content
* Fast and modern frontend experience

---

## Getting Started

1. Clone the repository
   `git clone https://github.com/sakg-dev/the-climate-note.git`

2. Navigate to the project
   `cd the-climate-note`

3. Install dependencies
   `npm install`

4. Run the development server
   `npm run dev`

Open http://localhost:3000 in your browser.

---

## Build for Production

`npm run build`

`npm start`
